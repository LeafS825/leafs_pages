# 🎉 网站诞生记

**2026年3月21日**，这个网站正式诞生了！

## 技术栈

这个网站基于 **VuePress** + **vuepress-theme-plume** 构建，部署在 Cloudflare Workers 上。特别感谢：

- [VuePress](https://vuepress.vuejs.org/) - 静态网站生成器
- [vuepress-theme-plume](https://theme-plume.vuejs.press/) - 精美的主题
- [PixelBlast](https://github.com/pengzhanbo) - 炫酷的背景效果
